{
  - 1️⃣ What is the difference between var, let, and const?
1  Ans: var → function-scoped variable that can be redeclared and reassigned.
        let → block-scoped variable that can be reassigned but not redeclared in the same scope.
        const → block-scoped variable whose value cannot be reassigned after declaration.
}

{
      - 2️⃣ What is the spread operator (...)?
2  Ans:  The spread operator is a JavaScript operator used to expand or unpack elements of an array or properties of an object into another array or object.
}

{
      - 3️⃣ What is the difference between map(), filter(), and forEach()?
3  Ans:   map() → creates a new array by applying a function to every element.
          filter() → creates a new array containing elements that satisfy a condition.
          forEach() → executes a function for each element in an array but does not return a new array.
}


{
      - 4️⃣ What is an arrow function?
4 Ans:    An arrow function is a shorter syntax for writing functions in JavaScript, introduced in ES6, using the => symbol.
}


{
      - 5️⃣ What are template literals?
5 Ans:    Template literals are string literals in JavaScript that allow embedded expressions and multiline strings, written using backticks ( ) instead of quotes.
